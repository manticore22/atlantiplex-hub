Design System Playground

What it is
- A tiny Storybook-like playground to validate visuals as you extend components.
- Features: simple hash-based routing to toggle previews; lightweight, CSS-only visuals using your design-system tokens.

How to run
- Open `design-system-playground/index.html` in a browser.
- Use the left navigation to switch between component previews.
